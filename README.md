# Telegrambot_sport
Создание простого бота который будет выводить информацию об команде результаты их игар и т.д
# Для запуска проекта

1. Выполните команду   git clone https://github.com/Reksiyaroslav/Telegrambot_sport.git 
2. Для создание виртуального окружение
    python -m venv .venv
    .venv\Scripts\activate  
3.  Установка зависимости: 
    pip install -r requirements.txt
4. Создайте .env похожий на .env.example и создайте свой токен для бота это можно сделать @BotFather в телеграмме 
5. Запустите файл run.py и Всё будет работ.
